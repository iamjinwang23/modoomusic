# MONO 알림·토스트 문구 카탈로그

> 앱/웹에서 사용자에게 노출되는 **푸시 알림 · 인앱 알림 · 토스트(스낵바)**의 실제 문구를 한곳에 모은 참고 문서.
> 문구를 바꾸려면 아래 "출처" 경로의 코드를 직접 수정해야 함(하드코딩).
> 최종 정리: 2026-07-24

---

## 1. 푸시 · 인앱 알림 (서버 발송)

서버가 이벤트 발생 시 **인앱 알림(notifications 테이블) + 웹/Expo 푸시**를 함께 보냄.
`{이름}`은 행위자 표시명(없으면 `누군가`), `{제목}`은 곡 제목, `{커뮤니티}`는 커뮤니티 이름.

### 개인 활동

| 상황 | 카테고리(토글) | 푸시 제목 | 푸시 본문 | 탭 시 이동 | 출처 |
|---|---|---|---|---|---|
| 곡 완성 | 곡 완성 | `곡이 완성됐어요` | `'{제목}' 곡을 지금 들어보세요.` | 해당 곡 재생 | `app/api/generate/route.ts` |
| 영상 완성 | 곡 완성 | `영상이 완성됐어요` | `영상 커버가 완성됐어요. 지금 확인해보세요.` | 해당 곡 재생 | `services/video-finalize.service.ts` |
| 영상 실패(환불) | 곡 완성 | `영상 만들기에 실패했어요` | `영상 생성에 실패해 환불 처리했어요.` | 홈 | `services/video-finalize.service.ts` |
| 내 곡에 좋아요 | 좋아요 | `새 좋아요` | `{이름}님이 회원님의 곡을 좋아했어요` | 해당 곡 재생 | `app/api/songs/[id]/like/route.ts` |
| 내 곡에 댓글 | 댓글·답글 | `새 댓글` | `{이름}님이 회원님의 곡에 댓글을 남겼어요` | 해당 곡 재생 | `app/api/songs/[id]/comments/route.ts` |
| 내 댓글에 답글 | 댓글·답글 | `새 답글` | `{이름}님이 회원님의 댓글에 답글을 남겼어요` | 해당 곡 재생 | `app/api/comments/[id]/reply/route.ts` |
| 나를 팔로우 | 팔로우 | `새 팔로워` | `{이름}님이 회원님을 팔로우했어요` | 크리에이터 프로필 | `app/api/profiles/[id]/follow/route.ts` |
| 크레딧 충전 완료 | 크레딧 충전 | `크레딧이 충전됐어요` | `충전한 크레딧으로 새 곡을 만들어보세요.` | 설정 | `services/payment.service.ts` |

| 상황 | 카테고리(토글) | 푸시 제목 | 푸시 본문 | 출처 |
|---|---|---|---|---|
| 내 글에 좋아요 | 커뮤니티 | `새 좋아요` | `{이름}님이 회원님의 글을 좋아했어요` | `services/community-post.service.ts` |
| 내 글에 댓글 | 커뮤니티 | `새 댓글` | `{이름}님이 회원님의 글에 댓글을 남겼어요` | `services/community-post.service.ts` |
| 내 댓글에 답글 | 커뮤니티 | `새 답글` | `{이름}님이 회원님의 댓글에 답글을 남겼어요` | `services/community-post.service.ts` |
| 폐쇄 예고 | 커뮤니티 | `커뮤니티 폐쇄 예고` | `'{커뮤니티}' 커뮤니티가 {N월 N일}에 폐쇄돼요. 그 전에 내가 작성한 글을 내보낼 수 있어요.` | `services/community.service.ts` |
| 새 가입 신청(매니저) | ⚠️ 없음 | `새 가입 신청` | `'{커뮤니티}'에 새 가입 신청이 있어요.` | `services/community-join.service.ts` |
| 가입 승인 | ⚠️ 없음 | `가입이 승인됐어요` | `'{커뮤니티}' 커뮤니티 가입이 승인됐어요.` | `services/community.service.ts` |
| 가입 거절 | ⚠️ 없음 | `가입이 거절됐어요` | `'{커뮤니티}' 커뮤니티 가입이 거절됐어요.` (+ `사유: {사유}`) | `services/community.service.ts` |
| 내 글 삭제됨(모더레이션) | ⚠️ 없음 | `게시글이 삭제되었어요` | `'{커뮤니티}' 커뮤니티에서 회원님의 게시글이 삭제되었어요.` | `services/community-post.service.ts` |
| 커뮤니티에서 내보내짐 | ⚠️ 없음 | `커뮤니티에서 내보내졌어요` | `'{커뮤니티}' 커뮤니티에서 내보내졌어요.` | `services/community.service.ts` |

> ⚠️ **카테고리 없음**: 가입/모더레이션 계열 푸시는 카테고리 인자 없이 발송됨 → **항목별 "커뮤니티" 토글로 끌 수 없고**, 오직 "전체 알림"(마스터) 끄기로만 차단됨. 단, 인앱 알림 목록에서는 `community` 카테고리로 분류됨(표시상 불일치). 향후 커뮤니티 카테고리로 게이팅 통일 검토 대상.

## 4. 알림 설정 구조 (토글 ↔ 알림 매핑)

설정 → **알림 설정**(뎁스 화면). 상단 **전체 알림**(마스터) + 아래 **항목별 토글**.
서버는 `notification_preferences` 테이블로 관리 (마이그레이션 063).

- **전체 알림 = `push_enabled`** — off 시 개별 값은 **유지한 채** 모든 푸시 차단. on 시 개별 설정 복원.
- 항목별 토글: 마스터 off면 비활성(회색) 표시.

| 토글 항목 | 카테고리 키 | 묶이는 알림 타입 |
|---|---|---|
| 곡 완성 | `song_complete` | song_complete (곡·영상 완성/실패) |
| 좋아요 | `likes` | like |
| 댓글·답글 | `comments` | comment |
| 팔로우 | `follow` | follow |
| 커뮤니티 | `community` | community_like, community_comment, community_closing, community_join_request/approved/rejected |
| 크레딧 충전 | `credit` | credit_charged |
| 공지 | `announcement` | system (공지·운영 알림) |

> 매핑 정의: `packages/shared/src/domain/index.ts` — `PUSH_CATEGORIES`, `PUSH_CATEGORY_LABELS`, `notificationTypeToCategory()`.
> 게이팅 로직: `apps/web/services/push.service.ts` — `sendPushToUser`(마스터+카테고리 체크), `sendPushToAll`(제외 셋 필터).
